# Himanshu Umbrel Apps

Community Umbrel app store by `singhsona27`.

## Apps

- `singhsona27-dgb-solo` - DGB Solo Node +Mining Pool
- `singhsona27-bch-solo` - BCH Solo Node +Mining Pool
- `singhsona27-bsv-solo` - BSV Solo Node +Mining Pool
- `singhsona27-karashare` - Karashare
- `singhsona27-kara-hosting` - Kara Hosting
- `singhsona27-etc-solo` - ETC Solo Node + Pool

## Add This Store To Umbrel

Use this repository URL:

```text
https://github.com/singhsona27/himanshu-umbrel.git
```

After adding the store in Umbrel, install the app you want and use the
dashboard settings or app installer pages described in each app README.
