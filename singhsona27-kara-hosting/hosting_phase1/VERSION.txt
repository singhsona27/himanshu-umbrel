Karacraft Hosting V5

V6 foundation additions:
- Website-centric customer dashboard at /websites.php
- Per-website management panel at /website.php
- Tabs for Overview, Domains, Files, Databases, Backups, Security, Performance, Apps, Logs, and Support
- One-click installers for WordPress, Moodle, OpenCart, Joomla, Drupal, and Grav
- Customer Fresh Start action to wipe site files, app data, and reset the site database before reinstalling an app
- Custom customer PHP image for app hosting with Sodium, Intl, ZIP, GD, SOAP, MySQL extensions, Apache rewrite, higher PHP limits, and unrestricted open_basedir for Moodle/OpenCart compatibility
- Customer usability improvements: flash success/error messages for button actions, guided 3-step custom-domain setup, clearer Cloudflare DNS instructions, and ZIP upload/extract completion notices
- Final usability pass: File Manager password reset in customer Files tab, custom domains shown as primary website URLs after verification, screenshot-style Cloudflare domain guide, open-website prompts after ZIP extract and app install, and hardened domain verify/repair routing support

V5 builds on V3.1 with the bug fixes and platform upgrades discussed for the next professional release.
